    Enables HAR recording for all pages into recordHar.path file. If not specified, the HAR is not recorded. Make sure to await CloseAsync(Boolean) for the HAR to be saved. You can use HarResult to deserialize the generated JSON file.
    RecordVideoOptions recordVideo
